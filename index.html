DailyLinks - a Chrome extension

Download Download View dailylinks-chrome on GitHub

DailyLinks is a Chrome extension that makes it easy to open those websites you visit every day, and supports different links per day (good for webcomics).

Installation

Install from the Chrome Web Store.

Configuration

Open up the Options page for the extension to configure your schedule. Save the link at the top of the page to access later to view your links for that day, and to open them all at once in separate tabs.
